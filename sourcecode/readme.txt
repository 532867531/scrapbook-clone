Where is the soure code?


If you are looking for the source code of ScrapBook Plus (SBP), you can find it inside scrapbook.jar.
If you are looking for the source code of ScrapBook 1.3.3.7 (SB), on which SBP is based on, download SBP 1.6.12.22. You can find the files inside a folder named backup and scrapbook.jar. All files with a date of June 26, 2008 or before are identical with the source of SB.
